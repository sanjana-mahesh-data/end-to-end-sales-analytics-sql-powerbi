# End-to-End Sales Analytics (SQL + Power BI)

## 📊 Project Overview
This project involves designing a relational database and performing SQL-based data analysis to extract insights, followed by visualization using Power BI dashboards.

## 🎯 Objective
- Perform data extraction and transformation using SQL
- Analyze sales performance and regional trends
- Develop dashboards to support business decision-making

## 🛠 Tools & Technologies
- SQL Server
- Power BI
- DAX
- Power Query

## 📂 Dataset
- Large-scale sales dataset (205M+ records) including revenue, profit, and regional data

## 🔍 SQL Analysis
- Used JOINs to combine multiple tables
- Performed aggregations (SUM, AVG, COUNT) for KPI calculation
- Applied filtering and grouping for trend analysis

## 📊 Dashboard Features
- Sales and profit analysis
- Regional performance insights
- KPI tracking and trend visualization

## 🚀 Outcome
Generated actionable insights to improve visibility into business performance and support data-driven decision-making.
